How to Play in LAN Mode

  1. Host: Run the server using npm start in the terminal
  2. Host: Connect to http://localhost:3000 in your browser
  3. Players: Connect to the host's IP address (e.g., http://192.168.1.100:3000)
  4. Everyone: Create or join a table and start playing!
